# Installation

<Intro>
Jumppad runs on macOS, Linux and Windows (natively and with WSL2).
</Intro>

## Prerequisites

To use Jumppad you need a recent version of Docker and the Git CLI installed on your system.

- [Docker](https://docs.docker.com/)
- [Git](https://git-scm.com/)

## Quick install (Linux, Mac and Windows WSL2)

The quick install script allows you to easily install the latest version of Jumppad. Run the following command in your terminal, the install script will prompt you for administrator access if required.


## Using a package manager

Jumppad is published as a package for Linux (apt, yum), macOS ([Homebrew](https://brew.sh/)) and Windows ([Chocolatey](https://chocolatey.org/)) and can be installed through their respective package managers.

<CodeGroup>
```shell {{ title: 'macOS' }}
brew install jumppad-labs/homebrew-repo/jumppad
```

```shell {{ title: 'Windows' }}
choco install jumppad
```

```shell {{ title: 'Debian' }}
echo "deb [trusted=yes] https://apt.fury.io/jumppad/ /" | \
  sudo tee -a /etc/apt/sources.list.d/fury.list

sudo apt-get update

sudo apt-get install jumppad
```

```shell {{ title: 'Fedora' }}
echo "[fury] name=Gemfury Private Repo baseurl=https://yum.fury.io/jumppad/ enabled=1 gpgcheck=0" | \
  sudo tee -a /etc/yum/repos.d/fury.repo

yum install jumppad
```
</CodeGroup>
